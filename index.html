JavaScript Lessons
introduction
JavaScript mainly is used to add functionality to Websites. The history is interesting, but that won't be discussed here as the focus is on coding.

SOME THINGS JAVASCRIPT CAN DO:

Manipulate the style of HTML elements (DOM manipulation) Perform processing on data and other site specific variables Check form data before it's submitted Add and remove HTML elements from the DOM

HOW DOES JAVASCRIPT 'RUN'?

When JavaScript is part of Website, it is downloaded into the local machine memory and an interpreter in the browser 'runs' the JavaScript code are two phases involved in the execution of the JavaScript code on the local machine Phase 1 one is for the compilation of the code, where variables and functions are assessed Phase 2 is the actual interpretation of the code by the JavaScript 'engine' Each browser has its own JavaScript 'engine', for instance Chrome uses an open source module called V8 (the V8 source code was used in the development of the NodeJS platform)
